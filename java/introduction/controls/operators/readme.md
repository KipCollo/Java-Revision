# operators

## Comparison Operators
Used when comparing values.It includes:
 - == Equality-Returns true if equal 
 - != Inequality- Returns true if unequal
 - <= Less than or equal to

## Logical Operators
 Used when combining two or multiple boolean expressions.
 It includes:
 - && Logical AND
 - || Logical OR
 - ! Logical NOT
